# Force two (H, W, C) arrays to share the same channel count

Force two (H, W, C) arrays to share the same channel count

## Usage

``` r
stitch_match_channels(a1, a2)
```

## Arguments

- a1, a2:

  Numeric `(H, W, C)` arrays.

## Value

A list of the two arrays expanded to a common channel count.
